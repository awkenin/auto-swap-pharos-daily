# FEATURE BOT
#### - AUTO SWAP DAILY

### 1. Clone Repository

```bash
git clone https://github.com/awkenin/auto-swap-pharos-daily.git && cd auto-swap-pharos-daily
```

### 2. Install
```
npm install
```

### 3. Input private key
```
nano .env
```

### 4. run

```bash
node bot.js
```
